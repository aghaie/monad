# Q4 — Minimal Generative Core Report (Phase ΩΣ)

**Method:** `foundational-questions-1.0` · corpus-only · deterministic.

## 1. Objective
What is the smallest irreducible informational core capable of reconstructing the maximum amount of Quranic
structure?

## 2. Method
The minimal core is identified as the set of units whose removal least degrades coverage — operationally the
highest document-frequency roots (the hub), cross-checked against the Phase Ξ representation-collapse result
and the essentiality analysis (Q13).

## 3. Results
The minimal core is the **high-frequency hub**: اله، قول، كون، ربب، علم، امن، قوم، اتي، كفر، بين (top
document-frequency roots). These few anchors carry most of the discourse coverage; the remaining ~80% of
specific content is the irreducible lexical residual (representation-invariant: root 0.796 / lemma 0.751 /
word 0.724).

## 4. Interpretation
The minimal generative core is **not a deep semantic kernel** — it is the frequency hub. Reconstruction
beyond it fails (Phase P: structure does not beat frequency held-out). So the "core" compresses coverage but
**not** the specific content, which is irreducible.

## 5. Falsification Attempts
The hub's primacy survives the frequency null (Phase 17), held-out prediction (Phase P), and representation
collapse (Phase Ξ). The claim that a *smaller-than-hub* semantic core exists is falsified by the ~80%
residual.

## 6. Limitations
"Core" here is coverage-minimal, not meaning-minimal; the latter is unrecoverable structurally.

## 7. Conclusion
**The minimal core is the frequency hub.** It maximizes coverage but leaves ~80% of content irreducible —
the same information-theoretic core that survived every prior control.
