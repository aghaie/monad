# Q3 — Human-Model vs World-Model Report (Phase ΩΣ)

**Method:** `foundational-questions-1.0` · corpus-only · deterministic.

## 1. Objective
Does the Quran allocate more structural attention to a **model of the external world** or a **model of the
human being**?

## 2. Method
Reference frame from morphology (`features_raw` person codes): 1st + 2nd person = human **address** frame;
3rd person = **world description** frame. Counts are exact censuses of person-marked tokens.

## 3. Results
| frame | tokens | share |
|---|--:|--:|
| 1st person | 6,347 | |
| 2nd person | 11,522 | |
| **address (1st+2nd)** | **17,869** | **40.5%** |
| **described (3rd)** | **26,223** | **59.5%** |

## 4. Interpretation
The corpus is **a 3rd-person world delivered to a 2nd/1st-person human addressee**. 3rd-person reference
dominates the *content* (59.5%), but the 2nd-person address load (11,522 tokens) is far heavier than neutral
narration would carry — this is a text that continually turns to address a human listener. Neither frame
reduces to the other.

## 5. Falsification Attempts
These are exact corpus censuses, not co-occurrence inferences — there is nothing to falsify; the ratio is a
descriptive fact (stable by construction).

## 6. Limitations
Person frame ≠ animacy. "Human" vs "world" cannot be separated by animacy structurally; the address/described
split is the defensible proxy.

## 7. Conclusion
**BOTH — world-content in a human-address frame.** The Quran is not primarily one or the other: it models a
3rd-person world (59.5%) but is pervasively structured as direct human address (40.5%).
